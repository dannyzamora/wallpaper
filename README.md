# Auto WallPaper 
```
Using beautifulsoup to scrape images from 'https://www.wallpapermaiden.com', then setting that image as the wallpaper
```


### Prerequisites
```
Python3
Libs:
OS
ctypes
BeautifulSoup, SoupStrainer
urllib
ssl
```


### Instructions 

In CMD Pront run python3 wallpaper.py
